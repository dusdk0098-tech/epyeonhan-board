Windows desktop application

사진 대장 편집을 더 빠르고 단정하게.

PEDIT은 현장 사진 정리, 대장 작성, 출력물 검토를 돕는 Windows용 업무 프로그램입니다. 사진과 작업 데이터는 사용자의 PC에서 처리되며, 계정 로그인은 Google과 Supabase Auth를 통해 제공됩니다.

최신 설치 파일 개인정보처리방침

업무용 편집

사진 위에 대장 정보를 정리하고, 출력에 맞는 결과물을 만드는 데 집중한 도구입니다.

소셜 로그인

Google 로그인을 사용해 계정과 구독 상태를 확인합니다. 로그인 처리는 Supabase Auth를 통해 진행됩니다.

자동 업데이트

배포된 Windows 설치 파일과 업데이트 정보는 GitHub Releases와 GitHub Pages를 통해 제공됩니다.
